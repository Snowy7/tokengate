Tokengate + Vite

Client-safe VITE_* variables injected at build time via @tokengate/env-vite. No .env file on disk.

VariableValue
How this works: The Vite plugin loads env vars from Tokengate cloud at dev/build time. Variables prefixed with VITE_ are injected into import.meta.env (client-safe). Server-only vars (DATABASE_URL, API_KEY) are in process.env during SSR only.
